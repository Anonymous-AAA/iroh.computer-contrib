# Iroh Swift SDK

Iroh in your iOS or MacOS app! {{ className: 'lead' }}

Iroh is purpose-built for mobile.

## Getting Started

Install [`the iroh swift package`](https://pypi.org/project/iroh) with [pip](https://pip.pypa.io/en/stable/):

```bash
$ pip install iroh
```

## Learning the API

<Note>
  The Swift SDK is missing package documentation, the best reference to use for now is the [API documentation](/docs/api)
</Note>

The [API documentation](/docs/api) is the best place to learn about the API. Each command includes a swift example to help build your understanding of how iroh works, which you can compose together to meet your needs.
