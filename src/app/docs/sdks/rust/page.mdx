# Iroh in Rust

The reference implementation iroh is written in rust. {{ className: 'lead' }}

The iroh crate defines both the iroh library and iroh console. To work with iroh, add the following dependencies in your `Cargo.toml` file:

```toml {{ title: 'Cargo.toml' }}
[dependencies]
iroh = "0.12.0"
```

Iroh is built with the `tokio` runtime, and includes numerous feature flags that customize its behavior. See the [rust API documentation](https://docs.rs/iroh/) for more information.

## Learning the API

The [API documentation](https://docs.rs/iroh/) is the best place to learn about the API. Each command includes a rust example to help build your understanding of how iroh works, which you can compose together to meet your needs.
